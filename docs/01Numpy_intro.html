I noticed your attempt, but while it’s not incorrect, it’s not the most efficient approach.

What is NumPy?
NumPy, short for Numerical Python, represents a cornerstone in Python’s library ecosystem. This fundamental library is tailored for handling an array of numerical and mathematical operations with precision and efficiency. Its prowess lies in furnishing robust support for arrays, matrices, and n-dimensional arrays, granting users a versatile playground for managing complex data structures effortlessly.

Beyond its array-handling capabilities, NumPy offers a comprehensive suite of mathematical functions specifically engineered to operate seamlessly on these data structures. These functions not only enhance computational speed but also provide an extensive toolkit for performing intricate mathematical operations with ease. As a result, NumPy serves as a powerhouse for numerical computations and facilitates a diverse array of scientific and mathematical applications within the Python programming environment

Number of ways to make NumPy arrays

Methods to Create NumPy Arrays

Description

From Lists or Tuples:

np.array([1, 2, 3, 4, 5])

Creates a NumPy array from a Python list.

np.array((1, 2, 3, 4, 5))

Generates a NumPy array from a Python tuple.

Using np.arange():

np.arange(10)

Produces an array with evenly spaced values from 0 to 9.

Using np.zeros() and np.ones():

np.zeros((3, 4))

Constructs a 3x4 array filled with zeros.

np.ones((2, 2, 2))

Generates a 2x2x2 array filled with ones.

Using np.linspace():

np.linspace(0, 10, 5)

Generates an array of 5 evenly spaced values from 0 to 10.

Using np.random module:

np.random.rand(2, 3)

Generates a 2x3 array with random values between 0 and 1.

np.random.randint(1, 11, (3, 3))

Produces a 3x3 array with random integers from 1 to 10.

From Existing Arrays:

np.copy(existing_arr)

Creates a copy of an existing NumPy array.

np.ones_like(existing_arr)

Generates an array of ones with the same shape and type as an existing array.

Converting Python sequences:

np.array(list_or_tuple)

Converts Python lists or tuples into NumPy arrays.

Built-in functions:

np.arange(start, stop, step)

Creates an array with values from start to stop with a specified step size.

np.linspace(start, stop, num)

Generates an array with num evenly spaced values between start and stop.

np.zeros(shape)

Generates an array of zeros with the specified shape.

np.ones(shape)

Constructs an array of ones with the specified shape.

np.empty(shape)

Creates an uninitialized array with the specified shape.

Replicating/joining arrays:

np.concatenate((array1, array2))

Joins arrays together along a specified axis.

np.repeat(array, repeats)

Repeats elements of an array.

np.tile(array, reps)

Constructs an array by repeating the specified array.

Reading from files:

np.loadtxt('file_path')

Loads data from a text file.

np.genfromtxt('file_path')

Loads data from a text file with missing values handled.

Creating from raw bytes:

np.frombuffer(buffer_object, dtype)

Creates an array from a buffer object interpreted according to the specified data type.

Special library functions:

np.random.rand(shape)

Generates an array with random values in the given shape.

np.random.randint(low, high, size)

Generates an array of random integers between low and high in the specified size.

np.eye(N)

Constructs an identity matrix of size N.

np.diag(diagonal)

Generates a diagonal matrix from the provided diagonal values.
